Latest Project

Bioreactor with
Web-Dashboard

This project creates an automated bioreactor control system that monitors and regulates environmental parameters like pH, dissolved oxygen, and temperature. An ESP32 collects real-time sensor data, and a React-based dashboard provides an intuitive interface for monitoring, visualization, and control of the system.

Robotic Arm (ROS2)

This project focuses on a robotic arm controlled via ROS2 using Micro-ROS on an ESP32. The arm follows commands received through ROS2 topics, ensuring smooth integration with robotic frameworks. It enables precise real-time movement execution for automation, research, and industrial applications.

Automatic Pressure and Temperature Control

This project automates pressure and temperature regulation using a PID controller with two microcontrollers. One controls a servo motor based on an HX711 load cell, while the other handles wireless commands. The system dynamically adjusts to fluctuations, ensuring stable performance for industrial and laboratory applications.

3-Axis Gantry System

This project is a CNC-style gantry system controlled via G-code for precision motion. It utilizes a microcontroller with the AccelStepper library to drive stepper motors along the X, Y, and Z axes. It has Auto homing and it gives precise movement with 1mm resolution.

PCB Design

In this project I have designed extensible PCB shield for ESP32 which can be used for various applications like BLDC, Stepper motor control, etc. It has RS485, CAN, I2C, SPI, UART, GPIO, etc. interfaces. Also built in Rotary encoder interface and Temperature sensor.
